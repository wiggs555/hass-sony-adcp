# Sony ADCP Projector

Control Sony home cinema projectors over **ADCP** (Advanced Display Control Protocol) on TCP port 53595.

## Supported hardware

Tested with the **Sony VPL-VW295ES**. Other Sony projectors that expose ADCP should work, but command availability varies by model.

## Highlights

- Power and HDMI input via `media_player`
- Picture presets, aspect ratio, gamma, color temperature, and Motionflow
- Lens memory selection
- Color calibration controls
- Raw ADCP command service for advanced automations
- Optional SDAP discovery during setup

See the [README](https://github.com/wiggs555/hass-sony-adcp#readme) for setup, entity list, and automation examples.
